# Nettoyage Express

Site vitrine statique pour une entreprise de nettoyage professionnel : présentation des
prestations (vitres, bureaux, chantiers, moquettes, décapage), zones d'intervention et contact.

## Stack
HTML / CSS / JavaScript, sans dépendance ni build. Déployable tel quel (GitHub Pages, Vercel, S3…).

## Lancer en local
Ouvrir `index.html` dans un navigateur, ou servir le dossier :

```bash
python -m http.server 8000
```
